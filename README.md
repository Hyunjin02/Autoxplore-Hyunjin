# Autoxplore-Hyunjin
신한투자증권 프로 디지털 아카데미 프론트 프로젝트-Autoxplore<br/> 
<br/> 
<h3>Autoxplore는 차량 구매 사이트입니다.<br/> </h3>

html,css를 활용해 반응형 웹페이지로 구현했습니다.<br/>
모바일뷰, 태블릿뷰, PC뷰 사이즈에 반응하여 구성 레이아웃이 달라지는 것이 특징입니다.</br>

html,css로만 작성된 코드이기 때문에, 동적인 기능은 따로 구현되어 있지 않습니다.<br/> 
팀 프로젝트 중 제가 만든 2개의 페이지만 Git을 이용해 배포하였습니다.<br/>
